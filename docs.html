FSCSS Documentation

Explore all shorthand powers of FSCSS with real examples.

Variables ($var: ...)

FSCSS lets you define variables using $name: value; and use them with $name!. This helps reduce repetition and improves readability.

fscss
$myColor: #3498db;
.box {
  background: $myColor!;
}
css 
:root{ --myColor: #3498db; }
.box {
  background: var(--myColor);
}

re(), str(), store() Replace

Use str(), re(), or store() to store reusable CSS property groups and reference them by name inside selectors.

fscss 
str(headerStyle, "font-size: 2em; font-weight: bold;")

header { headerStyle } .header { headerStyle }
css 
header { font-size: 2em; font-weight: bold; } .header { font-size: 2em; font-weight: bold; }

rpt() Repeat

The rpt(n, value) function repeats a value or text a specified number of times. Useful for visual indicators or generating content strings.

fscss 
p::after {

content: "rpt(5, 'Hey ')"; }
css 
p::after { content: "Hey Hey Hey Hey Hey "; }

copy() Extract parts of values

The copy(n, var) function allows value extraction and partial reuse from a defined string, helpful when slicing lists or values.

fscss 
p::after {

content: "blue,red copy(4,my-blue)"; color: $my-blue!; }
css 
p::after { content: "blue,red"; color: var(--my-blue); } :root { --my-blue: blue; }

mx() / mxs() Mixins

Use mx() and mxs() to quickly apply multiple properties with the same value. mxs uses a shared value string, while mx requires appending colons manually.

fscss 
.card {

mxs(width, height, max-height, max-width, min-width, min-height, '200px') } .box { mx(width, height, max-height, max-width, min-width, min-height, ': 200px;') }
css 
.card { width: 200px; height: 200px; max-height: 200px; max-width: 200px; min-width: 200px; min-height: 200px; } .box { width: 200px; height: 200px; max-height: 200px; max-width: 200px; min-width: 200px; min-height: 200px; }

Share Properties: %1 - %6, %i

Shared values are assigned to indexes like %1, %2, and so on. They can be injected directly into selectors using shorthand.

fscss 
div{%2(width, height[:200px;])}
css 
div {

width: 200px; height: 200px; }

$(attribute: value)

Shorthand for attribute selectors. Converts $(type: submit) into [type='submit'].

fscss 
$(type:submit) {

background: green; color: white; }
css 
[type='submit'] { background: green; color: white; }

Keyframes Compact

Write animations compactly using $(@keyframes name, selectors, [<timing>]). FSCSS will expand this into both animation rules and the actual keyframe block.

fscss 
$(@keyframes slideIn, .box, .card, &[3s linear infinite]) {

from { transform: translateX(-100%); } to { transform: translateX(0); } }
css 
.box, .card { animation: slideIn 3s linear infinite; } @keyframes slideIn { from { transform: translateX(-100%); } to { transform: translateX(0); } }

Vendor Prefixing (-*-)

Use the -*- prefix to automatically apply vendor-specific properties across -webkit, -moz, -ms, and -o.

fscss 
.box {

-*-transform: rotate(45deg); }
css 
.box { -webkit-transform: rotate(45deg); -moz-transform: rotate(45deg); -ms-transform: rotate(45deg); -o-transform: rotate(45deg); }
